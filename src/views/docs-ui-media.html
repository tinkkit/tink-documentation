Images

Responsive images

By default all images are set responsive.
If you don't want your image to be responsive, add the class non-responsive or give the image the desired width and height.

If you don't want your SVG image to be responsive, add the class non-responsive to the image and be sure to give it a width and height in the html code or in the source code of the image.
