Overview

Introduction

Tink is by default themeable. You can use one of the themes listed in the side navigation or fork one to start creating your own theme.

Usage

It's important to know that Tink themes aren't just extensions of the tink core, but rather complete separate builds. As a result it is enough to only require the theme CSS file. However, all components require the tink core, so if you use a tool like wiredep to manage your dependencies automatically you need to put a small override into your bower.json file:

"overrides": { "tink-core": { "main": [] } }
