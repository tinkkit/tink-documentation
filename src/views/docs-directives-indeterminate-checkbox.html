A checkbox input can only have the state checked or unchecked, but visually there's a third state, the indeterminate state. This can be useful when you work with nested checkboxes.

Our indeterminate checkbox directive gives you the possibility to create these nested checkboxes, where the child elements will influence the state of the parent checkbox.
