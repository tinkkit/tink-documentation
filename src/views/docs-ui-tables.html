Combined classes

Feel free to combine any of the aforementioned classes: .table-condensed, .table-striped and .table-bordered.

Responsive Tables

Basic responsive table

Create a simple responsive table by wrapping any table in .table-responsive to make them scroll horizontally on small devices (under 768px). When viewing on anything larger than 768px wide, you'll not see any differences in these tables.

Flip-scroll table

Wrap your tables in .table-flip-scroll and we'll make them mobile friendly by enabling horizontal scrolling for the appropriate screen sizes. The table heading will stay fixed. This type of table should not be used with table footers.
