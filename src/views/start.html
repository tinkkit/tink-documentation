Getting started

Core styling

To install the basic version of Tink (styling only), navigate to your project via the command line and run the following command:

bower install tink --save

Note that this command requires you to have Bower installed.

A complete overview of the styling elements can be found in the styling kit section.

Components

For now all Tink components rely on AngularJS and are built as separate directives. Every component can be installed separately via Bower.

Make sure to carefully read the instructions, because the installation process differs depending which component you're installing.

A complete list of all the directives can be found in the directives section.

Themes

If you prefer to use one of our themes instead of the default Tink styling, you can just replace the core with one of our themes.

Check out the themes section or use the theme switcher in the top right corner to see the themes in action.
