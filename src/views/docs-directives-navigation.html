Right navigation

Other implementations

Left navigation can always be closed

In order to permanently show the hamburger menu for the left menu.
<html class="has-nav-left nav-left-can-always-close"> … </html>
In order to permanently show the hamburger menu for the left menu and start with the left menu open.
<html class="has-nav-left nav-left-can-always-close-and-starts-open"> … </html>

Right navigation can always be closed

In order to permanently show the hamburger menu for the right menu.
<html class="has-nav-right nav-right-can-always-close"> … </html>
In order to permanently show the hamburger menu for the right menu and start with the right menu open.
<html class="has-nav-right nav-right-can-always-close-and-starts-open"> … </html>
