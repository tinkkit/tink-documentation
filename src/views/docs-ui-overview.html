Overview

Introduction

Tink is heavily inspired on the Bootstrap framework. We decided to use a lot of its functionality and to improve it where we thought we could.

Normally, a minimum of effort is needed to Tinkify your work. Most projects made with Bootstrap can be ported to Tink easily.

Industry standards

Tink makes use of the HTML5 doctype. Include it at the beginning of all your projects.

<!DOCTYPE html> <html lang="en"> … </html>

Using the Sass version of Tink

If you would inspect the Tink bower package you will also find the complete Sass version of Tink inside. You can use these files to compile Tink yourself or you can use some of the files to just use our variables, functions and mixins:

@import "../../bower_components/tink-core/dist/tink/variables"; @import "../../bower_components/tink-core/dist/tink/functions"; @import "../../bower_components/tink-core/dist/tink/mixins";
