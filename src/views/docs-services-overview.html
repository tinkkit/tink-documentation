Overview

Introduction

Tink services are substitutable objects that are wired together using dependency injection (DI). You can use services to organize and share code across your app.

For now, all Tink services are built for Angular. You can recognize this repositories on our GitHub page by their -angular suffix. Check the left navigation for a complete overview.

Basic usage

Most of these compservicesonents require nothing more than to install them via Bower and inject them as a dependency in your controller. The only thing more to do is include all necessary files in your project or let a tool like wiredep do its magic!
