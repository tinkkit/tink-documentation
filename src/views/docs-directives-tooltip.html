Popover Component

single datepicker

Examples

A tooltip left or a tooltip on the right and a tooltip at the bottom

Usage

To add a tooltip to a link element just add the "tink-tooltip" attribute to the element.

Options

Attr Type Default Details
data-tooltip-placement string top Where to place it, top, bottom, left, right.
data-tooltip-trigger string focus What should trigger the show of the popover? click, mouseenter, focus.
